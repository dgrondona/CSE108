## Python

### For Loop with Range

- For loops can iterate over a sequence of nmbers using the range
- Range returns a sequence of nmbers

```python
for x in range(5): # Prints out the numbers 0,1,2,3,4
    print(x)
for x in range(3, 6): # Prints out 3,4,5
    print(x)
for x in range(3, 8, 2): # Prints out 3,5,7
    print(x)
```

#### Inheritance

- Can define a class that inherits all the methods and properties from another class
- Parent Class (Base Class)
- Child Class

```python
class Student:

    def __init__(self, id, major, debt):
        self.id = id
        self.major = major
        self.debt = debt

    def gotPaid(self, amount):
        self.debt -= amount


class CSStudent(Student):

    def __init__(self, id, debt, langauges):
        self.id = id
        self.major = "CS"
        self.debt = debt
        self.languages = languages

    def paidDebt(self, amount): # overides the method from the parent
        self.debt -= 2 * amount

john = CSStudent("John", 1000, 7)
john.payDebt(100)
```

```python
class Student:

    def __init__(self, id, major, debt):
        self.id = id
        self.major = major
        self.debt = debt

    def gotPaid(self, amount):
        self.debt -= amount


class CSStudent(Student):

    def __init__(self, id, debt, langauges):
        super().__init__(id, "CS", debt) # inherited from parent
        self.languages = languages

    def paidDebt(self, amount):
        for i in range(2):
            super().paidDebt(amount)

john = CSStudent("John", 1000, 7)
john.payDebt(100)
```

### Dictionaries

- Very efficient data structure (hash table)
- Keys
    - used to find the corresponding value
    - must be unique and immutable
    - can be strings, numbers, or tuples
- Values can be of any type

```python
grades = {'John' : 'A',
    'Emily' : 'A+',
    'Betty' : 'B',
    'Mike' : 'C',
    'Ashley' : 'A'}

print(grades['Emily']) # get value

grades['John']= 'C-' # set value

grades['Jose'] = 'B+' # add a person

del(grades['John']) # remove a person

grades.pop['Betty'] # same thing as del

# prints all of the keys
for key in grades.keys():
    print(key)

# prints all of the values
for value in grades.value():
    print(value)

# prints all of the items
for items in grades.items():
    print(items)
```

### JSON

- JSON stand for JavaScript Object Notation
- JSON is very common web-based format for sending and recieving messages
- JavaScript and Python have easy ways to create objects and dictionaries from JSON
Python supports JSON with a built-in package called json

#### Dump / Dumps, Load / Loads

**Dump:**
```python
import json

grades = {'John' : 'A',
    'Emily' : 'A+',
    'Betty' : 'B',
    'Mike' : 'C',
    'Ashley' : 'A'}

with open("sample.json", "w") as outfile:
    json.dump(grades, outfile)
```

**Load:**
```python
import json

with open("sample.json", "r") as file:
    grades = json.load(file)
    print(grades)
```
